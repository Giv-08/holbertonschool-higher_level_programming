The ``my_list`` module

Using ``my_list``

First, import ``my_list`` from the ``1-my_list`` file:

    my_list = __import__('1-my_list')
    MyList = my_list.MyList

Examples:
---------
Add self keyword in argument:

    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> my_list.print_sorted()
    [1, 4, 2, 3, 5]
    [1, 2, 3, 4, 5]
    [1, 4, 2, 3, 5]
