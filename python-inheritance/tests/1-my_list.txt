The ``my_list`` module

Using ``my_list``

First, import ``my_list`` from the ``1-my_list`` file:

    >>> print_sorted = __import__('1-my_list').print_sorted

Examples:
---------
Add self keyword in argument:

    >>> print_sorted()
    [1, 4, 2, 3, 5]
    [1, 2, 3, 4, 5]
    [1, 4, 2, 3, 5]

Add any word in argument:

    >>> print_sorted(list)
    [1, 4, 2, 3, 5]
    [1, 2, 3, 4, 5]
    [1, 4, 2, 3, 5]
